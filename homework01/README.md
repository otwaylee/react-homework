# 과제1 - 디자인 -> 개발 핸드오프 과제
auto layout 실습으로 완성한 시안 핸드오프 
--------------------
## 과제 수행 과정
1. 순서가 필요없는 아이템 리스트로 `<ul>` 활용, 너비 320px 지정
2. `<img>`와 `<p>`를 `<li>`로 그룹화, 기본 스타일 제거
3. `<li>`아이템 
    - `display: flex` 의 row 방향으로 정렬 및 (교차축) 가운데 정렬
    - 아이템간 간격 `gap: 20px` , `padding: 12px`
4. `<ul>` 아이템 리스트
    - column 방향으로 아이템 정렬 
    - 아이템간 간격 `gap: 4px`
